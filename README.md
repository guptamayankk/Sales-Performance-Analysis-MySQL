# 📊 Sales Performance Analysis Using MySQL

## 📌 Project Overview

This project analyzes retail sales data using MySQL to generate business insights. The analysis covers customer behavior, product performance, revenue trends, and sales performance using SQL queries.

The project follows a **star schema** data model consisting of one fact table and two dimension tables.

---

# 🎯 Business Problem

A retail company wants to monitor sales performance and answer important business questions such as:

- Which products generate the highest revenue?
- Which customers spend the most?
- Which product categories perform best?
- How do monthly sales change over time?
- Which customers should be considered VIP customers?

SQL is used to answer these questions and support business decision-making.

---

# 📂 Dataset

The project contains three datasets.

| Table | Description |
|--------|-------------|
| fact_sales | Sales transaction details |
| dim_customers | Customer information |
| dim_products | Product details |

---

# ⭐ Data Model

The project follows a **Star Schema**.

```
                dim_customers
                     │
                     │
                     │
dim_products ─── fact_sales
```
